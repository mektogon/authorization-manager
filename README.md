# Authorization Manager
Микросервис для авторизации пользователей.

## Технологии
Проект разрабатывался с использованием:

- Spring Framework (Core, Boot, Data JPA, HikariCP, Security, MVC)
- Lombok
- PostgreSQL
- Swagger
